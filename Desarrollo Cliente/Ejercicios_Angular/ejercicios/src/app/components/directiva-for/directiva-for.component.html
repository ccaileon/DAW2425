Ejercicio 1: Listado de productos

Directiva estructural for

Define un array de productos en el componente y muestra la lista en la vista usando *ngFor mediante cartas, donde se mostrarán todas las características del elemento. Para ello, es necesario que te definas un modelo de ingrediente, donde tiene como caracteristicas nombre, precio y tipo.

{{ producto.nombre }}
Precio: {{ producto.precio | number : "1.2-2" }}€

Este producto es de tipo: {{ producto.tipo }}
